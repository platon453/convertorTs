# Конвертер Валют

Этот проект представляет собой одностраничное приложение для конвертации валют (EUR в RUB) с разделением на Frontend и Backend.

## Структура проекта

- `/client`: Frontend приложение (Vite + TypeScript)
- `/server`: Backend приложение (Node.js + Express + TypeScript)

## Запуск проекта

### 1. Установка зависимостей

**Backend:**

Перейдите в директорию `server` и установите зависимости:

```bash
cd server
npm install
```

**Frontend:**

Перейдите в директорию `client` и установите зависимости:
```bash
cd client
npm install
```

### 2. Запуск

**Backend:**

Находясь в директории `server`, выполните команду:
```bash
npm run dev
```
Сервер запустится на `http://localhost:3001`.

**Frontend:**

Находясь в директории `client`, выполните команду:
```bash
npm run dev
```
Клиентское приложение запустится на `http://localhost:5173` (или на другом порту, если 5173 занят).

После запуска, откройте в браузере адрес, который будет указан в терминале для Frontend.
